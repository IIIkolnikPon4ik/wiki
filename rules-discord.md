# Правила Discord

## Общие правила

- **Уважение** — Относитесь к другим участникам с уважением. Оскорбления, токсичность и травля запрещены.
- **Спам** — Запрещается спамить сообщениями, эмодзи, упоминаниями и ссылками.
- **Реклама** — Реклама сторонних серверов и проектов запрещена без разрешения администрации.
- **NSFW** — Контент для взрослых строго запрещён в любом виде.
- **Личные данные** — Не публикуйте личные данные других людей без их согласия.

## Правила каналов

- Используйте каналы по назначению (читайте описание каждого канала).
- В голосовых каналах запрещено намеренно мешать другим игрокам.
- Ники должны быть читаемы и не нарушать правила.
