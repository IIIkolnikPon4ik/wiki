# Моды

## Обязательные моды

Для комфортной игры на Pon4ikMine рекомендуется установить следующие моды:

### Fabric API
- **Версия:** 1.20.1
- **Описание:** Базовый мод-загрузчик, необходим для всех остальных модов
- **Ссылка:** modrinth.com/mod/fabric-api

### Sodium
- **Версия:** 1.20.1
- **Описание:** Оптимизация рендеринга — значительный прирост FPS
- **Ссылка:** modrinth.com/mod/sodium

## Рекомендуемые моды

### Iris Shaders
- Поддержка шейдеров для красивой картинки
- Совместим с Sodium

### Mod Menu
- Удобное меню управления модами
- Показывает список установленных модов

### Lithium
- Оптимизация серверной логики
- Снижает нагрузку на процессор

### BetterF3
- Улучшенный экран F3 с цветной информацией
- Показывает TPS сервера

## Запрещённые моды

Следующие моды **запрещены** и могут привести к бану:

- **X-Ray моды** любого вида
- **Minimap с радаром игроков**
- **Killaura / Combat hacks**
- **Speed hacks**
- **Fly моды**
- **Автокликеры**

## Совместимость

Сервер работает на версии **Minecraft Java 1.20.1**.
Fabric поддерживается, Forge — нет.
